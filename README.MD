Hi team, I have completed the basic Node.js version of our learning support system.

The purpose of this project is to build a website where students can ask learning questions and receive simple explanations, examples, key takeaways, and practice questions. The system should guide students instead of giving direct assignment answers.

So far, I have created the backend logic using main.js and retrieval.js. The system can detect the topic of a question, search the TXT knowledge base files, retrieve relevant information, and generate a final student-friendly response. The knowledge base currently includes Programming, Database, and AI Prompting topics.

For the next stage, we need to turn this into a working website.

Frontend should focus on the user interface where students can type questions and view the answer clearly.

Backend should focus on converting the current command-line program into an API that the website can call.

Knowledge base/data should focus on adding more topics, improving keywords, and keeping the TXT files readable.

Testing should focus on checking whether the system gives the correct topic, retrieves the correct answer, and formats the final response clearly.

The expected final result is a working website where a student enters a question, the system detects the topic, retrieves the correct knowledge, and displays a clear learning response.
